# UI Foundry Section Studio

Standalone, self-contained Section component generator built for marketplace sale and future UI Foundry SaaS merge-readiness.

## Features

- Responsive size: max width, min height, breakpoint behavior, safe spacing tokens
- Composition: slots, landmarks, sticky/fixed behavior where native, content density
- Alignment: flex/grid placement, gap, inset, order, wrap, and visual rhythm
- Accessibility: semantic element, landmark label, reduced motion, readable contrast
- React-only export: preview, visible code, copy, and download are sourced from one payload.
- Local shared controls: no runtime imports from other component projects.

## Commands

```bash
npm install
npm run typecheck
npm run lint -- --quiet
npm run build
npm run dev
```
